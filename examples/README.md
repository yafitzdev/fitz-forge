# Examples

Progressive examples demonstrating fitz-forge usage.

| Example | Description | Prerequisites |
|---------|-------------|---------------|
| [01_quickstart.py](01_quickstart.py) | Queue a plan and read the result | Ollama or LM Studio running |
| [02_configuration.py](02_configuration.py) | Configure providers and tuning options | None (config file only) |
| [03_mcp_integration.py](03_mcp_integration.py) | Use fitz-forge as an MCP server in Claude Code | Claude Code installed |

## Running

```bash
# Install fitz-forge
pip install fitz-forge

# Run any example
python examples/01_quickstart.py
```
